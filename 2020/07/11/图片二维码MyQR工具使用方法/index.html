图片二维码MyQR工具使用方法

##图片二维码MyQR工具使用方法
①：安装好python
②：在cmd命令行里安装myqr : pip install myqr


③：用命令：“myqr -c -v20 -p lmy.jpg 123456 ” 生成二维码
-c 代表使用彩色
-v20代表边长控制
-p代表背景图片
lmy.jpg代表使用的图片
123456代表二维码包含的内容
