下载连接转换器

Veröffentlicht am 2020-07-13

title: 下载连接转换器
date: 2020-07-13 11:32:24
tags:


import base64
urll=input('请输入需要转换的连接：')
url = urll
#url='thunder://原始链接'
# 看见thunder://后面的字符串比较熟悉，对没错就是base64编码，去除thunder://
surl=url.lstrip('thunder://')
# base64转成二进制
burl=base64.b64decode(surl)
# 这里注意，得看原来的二进制转换编码是什么，我这里是GB2312的，有时是utf-8的，慢慢试常用的编码格式
# ISO-8859-1
# GB2312
# GBK
# GB18030
# UTF-16
# UTF-8
httpurl=burl.decode('GB2312')
httpurl=httpurl.strip('AAZZ')
print('转换连接成功')
print(httpurl)
# 最后拿到连接，直接去浏览器下载，速度也很快

图片二维码MyQR工具使用方法

Veröffentlicht am 2020-07-11 Bearbeitet am 2020-07-24

##图片二维码MyQR工具使用方法
①：安装好python
②：在cmd命令行里安装myqr : pip install myqr


③：用命令：“myqr -c -v20 -p lmy.jpg 123456 ” 生成二维码
-c 代表使用彩色
-v20代表边长控制
-p代表背景图片
lmy.jpg代表使用的图片
123456代表二维码包含的内容
